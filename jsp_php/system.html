1. PHP 와 JavaScript 용도

    html 에서 사용하던 css 를 밖으로 빼주듯 공통코드들을 묶어서 밖으로 빼주는 역활을 하는 동적 코드의 집합.
    동적 코드 : php, jsp
    정적 코드 : html, css, javascript

  2. PHP와 스크립트의 차이

    php는 소스보기 하면 코드가 보여지지 않음. (서버에서 처리되는언어)
    script는 소스보기 하면 코드가 보임. (웹에서 처리되는 언어)

  3. 웹 구동방식

    웹브라우저 -> 웹 서버 -> PHP엔진 -> 데이터베이스

  4. 디버깅 방식

    웹페이지 -> 웹페이지 콘솔
    apach2 폴더 -> htdocs -> Error Log 파일의 마지막줄 확인

  5. php 코드 웹출력 방법

    php폴더 -> php.ini열기 (편집기 아무거나 사용)
    "display_errors = off"를 검색
    off 부분을 on으로 수정
    그 다음 아파치 재실행 시켜줌
